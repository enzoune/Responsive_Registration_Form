# 📋 Multi-Step Responsive Form (Formulário Completo)

> Um formulário de cadastro robusto e visualmente organizado, desenvolvido com HTML5 e CSS3, focado em fornecer uma experiência de preenchimento intuitiva e profissional.

## 📖 Sobre o Projeto

Este projeto integra o meu portfólio de estudos em **Programação Full-Stack**. O objetivo foi construir um **Formulário Responsivo** capaz de coletar uma gama variada de informações (dados pessoais, contato e localização) de forma estruturada e limpa.

O foco do desenvolvimento foi a organização lógica dos campos, garantindo que o usuário consiga navegar pelas informações de maneira fluida tanto em desktops quanto em dispositivos móveis.

## ✨ Destaques e Diferenciais Técnicos

* **Estrutura Modular e Semântica:** Uso de seções (`section`) e tags de cabeçalho (`header`) para delimitar claramente o propósito do componente.
* **Variedade de Inputs:** Implementação de diversos tipos de entrada, incluindo `text`, `number`, `date`, `radio` e `select`, demonstrando domínio sobre os elementos nativos do HTML5.
* **Layout Flexível com Colunas:** Uso de classes de coluna (`column`) para agrupar campos relacionados, como telefone/data de nascimento e cidade/região, otimizando o espaço em tela.
* **Validação Nativa:** Inclusão do atributo `required` em campos críticos de endereço, garantindo a integridade mínima dos dados antes do envio.
* **Interface de Gênero Customizável:** Organização de opções de rádio (`gender-box`) para uma seleção rápida e clara de perfil.
* **Organização Profissional:** Separação estrita entre a marcação estrutural (`index.html`) e a identidade visual localizada no diretório `css/style.css`.

## 🛠️ Tecnologias Utilizadas

* **HTML5:** Estruturação semântica, lógica de formulário e validação de campos.
* **CSS3:** Gerenciamento de layout (Flexbox/Grid), tipografia e design responsivo.
